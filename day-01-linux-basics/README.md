# Day 1 - Linux Basics

##  Objectif

Découvrir les bases de Linux et les commandes essentielles.

##  Ce que j’ai appris

* Naviguer dans le système de fichiers
* Comprendre les dossiers et fichiers
* Utiliser le terminal

##  Commandes utilisées

```bash
pwd
ls
ls -l
ls -a
cd /home
cd ..
mkdir test
touch file.txt
rm file.txt
```

##  Explications

* `pwd` : affiche le dossier courant
* `ls` : liste les fichiers
* `cd` : changer de dossier
* `mkdir` : créer un dossier
* `touch` : créer un fichier
* `rm` : supprimer
* `a` : lorsqu'on utilise ls -a permet de lister les fichiers cachés 
* `cd ..` : Permet de revenir au dossier parent


##  Ce que j’ai compris

Linux est basé sur un système de fichiers hiérarchique.
Le terminal permet de tout contrôler rapidement.
